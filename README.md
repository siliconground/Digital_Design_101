# 😁Digital Design 101
## 🫡 어서오세요
환영합니다!
우리의 삶 속 어디에나 있는 하드웨어를 Hardware Descriptive Language(HDL)언어 중 하나인 VerilogHDL을 사용하여 만들어보는 쌩기초 과정입니다. 누구나 HDL 언어를 활용하여 재미있고 기발하게 디지털 회로를 공부하고 자신만의 하드웨어를 만들어 볼 수 있어요!

## 구성은 다음과 같아요
1. **기초 Linux 익히기**
* ASIC개발 환경인 Linux를 운용하기 위한 최소한의 생존 Linux를 살펴봅니다! Digital IC의 세계에 들어온 이상 Linux는 필수죠. 여러분이 현업에 가서 퇴근을 빨리 하기 위한 Linux 기술을 익혀서 똑똑한 사원이 되어 보아요.
  
2. **HDL Modeling**
* 이 과정에서는 교수님마저도 사용하지 않는 시시콜콜한 HDL문법을 과감하게 생략해요. 그 보다는 최소한의 HDL도구를 가지고 좌충우돌 함께 모델링을 해 나가면서 필요한 레시피를 하나하나 엊어 보아요. LRM(Language Reference Manual)을 익히는 것은 덤이죠.

3. **Practical Modeling**
* HDL교과서에 나오는 심심한 예제 보다는 실제 사용되고 있는 칩의 Datasheet를 통해 Spec을 파악하고 HDL로 모델링 해 보아요. 키보드는 마우스보다 빠르다는 것을 잊지 마세요!

## 🔐 실무 개발환경을 고려하여 구성했어요
* ASIC이든 FPGA는 본인이 원하는 방향으로 실습을 할 수 있도록 개발 환경을 구성했어요. 상용 솔루션을 사용하여도 좋고, 혼자서 연습할때는 무료 소프트웨어를 사용해서 충분하게 즐겁게 학습할 수 있어요.

## 🛠️ 도구 모음 및 개발 환경은 다음과 같아요
 - [Cadence Xcelium](https://www.cadence.com/en_US/home/tools/system-design-and-verification/simulation-and-testbench-verification/xcelium-simulator.html) : (美) 케이던스사의 상업용 Logic Simulator
 - [Xilinx Vivado](https://www.xilinx.com/support/download.html) : (美) AMD사의 FPGA용 IDE
 - [EDA Playground](https://www.edaplayground.com/) : 웹 상에서 Logic Simulation을 할 수 있는 Open Source Page
 - [WSL](https://learn.microsoft.com/ko-kr/windows/wsl/install) : Windows상에서 Linux를 편리하게 운영할 수 있게 해주는 Hypervisor
 - [Visul Studio Code](https://code.visualstudio.com/) : 여러분의 수명을 눌려줄 똑똑한 Editor Program
 - [GIT](https://git-scm.com/) : 여러분의 프로젝트 관리와 협업을 도와줄 Source Control Management
- [GitHub](https://github.com/) : Git을 Clound Server에 운영하는 오픈소스 덕후들의 성지
